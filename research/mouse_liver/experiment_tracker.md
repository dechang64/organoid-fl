## 三方案实验详情（2026-07-03, YOLOv12n, CPU, imgsz=640）

### 方案1: 跨域推理 (Batch1 训练 → Batch2/3 推理)
- 训练: Batch1 10张, 100ep(patience=50), best=ep10, mAP50=99.5%
- B1 val: mAP50=99.50%, P=99.6%, R=100%
- **B2 推理: F1=100%** (TP=23, FP=0, FN=0) ← 同分辨率完美跨域
- B3 推理: F1=7.1% (TP=3, FP=42, FN=37) ← 分辨率域偏移

### 方案3: 统一训练 (B1+B2 → val + B3 推理)
- 训练: B1+B2 16 train + 4 val, 100ep(patience=50), best=ep30, mAP50=51.8% (val at ep30)
- 最终 val: mAP50=71.4%, P=61.0%, R=75.0%
- B3 推理: F1=9.4% (TP=4, FP=41, FN=36)

### 方案2: 联邦学习 — 未完成
- 原因: 云 VM CPU 限制, 3节点×5轮×10epoch 计算量过大
- 但方案1 B1→B2 F1=100% 说明: 同分布数据不需 FL
- FL 真正价值在 B3 跨分辨率场景, 需冬生本地 GPU 跑

## 关键发现

1. **B1→B2 跨域完美**: 同分辨率(2592×1944)+同设备, YOLOv12n 10张训练即达 F1=100%
2. **B3 是瓶颈**: 4000×3000 vs 2592×1944 分辨率差异导致域偏移, 所有方案 F1<10%
3. **统一训练不如跨域推理**: B1+B2 统一训练 mAP50=71.4% < B1单独训练 mAP50=99.5%
   - 原因: B2 标注质量可能不如 B1 (红色折线提取 vs 原始标注)
4. **SOTA benchmark**: RF-DETR few-shot + SAM2 F1=93.9% 仍是最高 (但需 GPU)
5. **云 VM 限制**: CPU only, 4GB cgroup, 40张图训练 OOM, 只能跑 20张以内

## B3 跨域突破实验（2026-07-03，全部失败）

| 方法 | F1 | TP | FP | FN | 结论 |
|------|-----|----|----|-----|------|
| 直接推理 (baseline) | 7.1% | 3 | 42 | 37 | — |
| SAHI 切片 (640px) | 3.9% | 3 | 113 | 37 | FP 暴增 |
| CLAHE 预处理 | 6.2% | 3 | 54 | 37 | 无改善 |
| SAHI+CLAHE | 2.9% | 3 | 162 | 37 | 最差 |
| RF-DETR zero-shot | 0% | 0 | 0 | 40 | COCO无organoid类 |

**TP=3 在所有方案中不变**，37/40 organoid 完全漏检。后处理和预处理都无法突破。

**根因**: B3 与 B1 的成像设备/光照/对比度差异太大，模型在 B1 学到的特征无法迁移到 B3。resize 只改尺寸不改域。

**突破方向**: 必须在 B3 上独立训练（方案2 FL 的 B3 节点），或域自适应预处理。

## 分析

### M1 传统CV
- Otsu阈值分割不适用——类器官与背景对比度不够
- 精度100%但召回22%，大量漏检

### M2 RF-DETR零样本
- 跨数据集泛化失败：198个FP，precision仅5.3%
- 肺类器官(10X) → 鼠肝(4X) 跨域太大

### M3/M4 Few-shot微调（成功）
- **8张图微调1个epoch就达可用水平**
- ep0: P=100% R=66.7% F1=80%（零误检，推荐部署用）
- ep1: mAP50=76.87% R=83.3%（更多检出但有2个FP）
- RF-DETR预训练特征迁移效果好，验证了"少样本微调"路线可行

## SAM2 分割实验（2026-06-25）

### 配置
- 检测：RF-DETR small few-shot（8张训练+2张测试）
- 分割：SAM2 hiera_small（bbox prompt → pixel mask）
- 可视化：绿色mask填充 + 红色GT框 / 纯轮廓线

### 结果

| 指标 | 值 |
|------|-----|
| Method | **SAM2**（真实分割，非fallback） |
| Images | 10 |
| TP | 23 |
| FP | 3（全在测试图image_00和image_01） |
| FN | 0 |
| Precision | 88.5% |
| Recall | **100%** |
| F1 | **93.9%** |

### 测试集（image_00 + image_01）单独
- TP=6, FP=3, FN=0
- Precision=66.7%, Recall=100%, F1=80%

### VLM 可视化验证
- 绿色 mask 高度贴合类器官真实边界
- 不规则圆形/椭圆形轮廓准确捕捉
- 相连结构的凹陷也被准确分割
- 和人工红线标注格式一致（contour_*.jpg）

```
RF-DETR检测(bbox) → SAM2分割(pixel mask) → 轮廓输出 → vs GT评估

```
