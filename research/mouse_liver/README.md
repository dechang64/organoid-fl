## 关键发现

1. **B1→B2 跨域完美 (F1=100%)**: 同分辨率+同设备，10张训练即完美泛化
2. **B3 是瓶颈**: 4000×3000 vs 2592×1944 分辨率域偏移，所有方案 F1<10%
3. **统一训练不如单批**: B2 标注质量（折线提取）可能不如 B1
4. **方案2 FL 未完成**: 云VM CPU限制，需冬生本地GPU
